## 所要時間
4時間くらい

## 利用したもの

### Nuxt.js
Vue.jsのフレームワーク

### @nuxtjs/content
MarkdownやJSONからNuxt.jsへのデータ受け渡しをやってくれるモジュール

### Tailwind CSS
UI Component

### Github
ソース管理

### Netlify
任意のGitHubリポジトリから静的サイトを生成して配信してくれるサービス

### Netlify Functions
FaaS: サーバレスでサーバサイド処理をしてくれるもの。中身はAWS Lambdaと同じ。
