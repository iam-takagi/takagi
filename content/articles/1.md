## 2. 利用技術

### Nuxt.js
Vue.jsのフレームワーク

###  @nuxtjs/content
MarkdownやJSONからNuxt.jsへのデータ受け渡しをやってくれるモジュール

### Tailwind CSS
UI Component

### Github
ソース管理

### Netlify
任意のGitHubリポジトリから静的サイトを生成して配信してくれるサービス
